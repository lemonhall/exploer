# 文件管理器 (File Explorer)

一个基于Rust和Druid框架开发的简单Windows GUI文件管理器。

## 功能特性
- 显示当前工作目录下的文件和文件夹列表
- 简洁的用户界面，支持滚动浏览
- 窗口尺寸800x600像素

## 技术栈
- Rust编程语言
- Druid GUI框架 (v0.8)
- Windows API绑定

## 构建与运行

1. 确保已安装Rust工具链
2. 克隆本项目
3. 在项目目录下运行:
   ```bash
   cargo run
   ```

## 项目结构
- `src/main.rs` - 主程序文件
- `Cargo.toml` - 项目配置和依赖
